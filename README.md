# education

基于 React + TypeScript + Vite + Ant Design 的幼教后台管理系统（纯前端）。

## 技术栈

- React 19
- TypeScript
- Vite
- Ant Design 5
- React Router 6
- Zustand
- ECharts
- Mock.js + localStorage

## 本地开发

```bash
npm install
npm run dev
```

## 质量检查

```bash
npm run lint
npm run build
```

## 功能概览

- 登录与角色权限控制
- 仪表盘与数据可视化
- 幼儿信息管理（含导出）
- 日常考勤管理
- 食谱拖拽与营养分析（支持导出图片）
- 成长档案管理
- 家园共育（发布通知、导出活动名单）
- 保健卫生管理
- 教职工管理
- 财务管理
- RBAC 权限配置
